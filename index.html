Photo Sort & Clean

Automate your RAW & JPG photo management workflow

Project Overview

This project aims to help photographers and content creators efficiently manage their photo collections by:

  • Automatically sorting RAW (.CR2) and JPG files into separate folders.
  • Cleaning RAW files that don't have a matching JPG counterpart.
  • Enabling safe photo archiving by moving unused files instead of deleting.
  • Preparing the ground for more advanced scripts: auto adjustments, flash detection, LUT styling, and more.

How to Use

Run the script and enter the root folder path where your photos are stored. The script will:

  • Create JPG and CR2 folders if they don't exist.
  • Move your files to the respective folders.
  • Move unmatched RAW files to a safe archive folder instead of deleting.
  • Print a summary of actions performed.

Scripts Available

  • Windows Version (Python)
  • Ubuntu/Linux Version (Python)
Warning: Always backup your photos before running scripts. These tools move files, so ensure you have copies in case of unexpected behavior.
